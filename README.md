# Environmental analysis of Radio-AGN in the COSMOS Field

This repository contains the work I did for my Master's research investigating the environmental surface densities for Radio-selected AGN in the COSMOS field. Examining how local galaxy environments change for different AGN populations, such as High-Luminosity AGN (HLAGN) and Mid-to-low Luminosity AGN(MLAGN)

## Repository Structure
- Data - Where I store my .FITS files of the catalogues used
- Code - All the code used for the project
- Graphs - Storage for the results produced in the project

## Requirements
The PYTHON libraries used are:
~~~
numpy
matplotlib
scipy
astropy
~~~
